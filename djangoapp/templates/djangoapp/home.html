Welcome to Optim'R

Règles de confidentialité:

Vos coordonnées et votre nom d’utilisateur sont des données privées enregistrées pour des raisons de sécurité. Vos données personnelles ne seront pas redistribuées ou vendues à un tiers.
Les données que vous envoyez sur le site web Optim’R ne sont pas conservées plus longtemps que le temps d’ouverture de la page web d’Optim’R. Cela veut dire qu’une fois que vous fermez votre page web avec le site web Optim’R, les données que vous avez téléchargées sont effacées de notre système.
Des fichiers logs, qui décrivent votre activité sur notre site web, sont enregistrés. Les données que vous envoyez sur notre serveur ne sont pas enregistrées dans ces fichiers logs. Ces fichiers logs sont requis pour assurer la sécurité du site web Optim’R.

Conditions d’utilisation:

Le présent site web n’est ouvert qu’aux utilisateurs qui souhaitent traiter des données qui n’identifient pas les personnes physiques. Aucune donnée nominative ou identifiante ne doit être contenue dans les données chargée vers la plateforme Optim’R : noms de personnes, adresses, adresses mail, numéro de sécurité sociale.
